Qu'est-ce que "Platform-Manager" ?

Platform-Manager est un système de Gestion de plate-forme technologique.

Platform-Manager a été conçu pour faciliter la gestion de plate-forme technologique. Il intègre des outils de gestion d'utilisateurs (appartenance, responsable, charte...), des outils de gestion et de réservation de ressources (salles, microscopes...), de gestion de services et de bases de données métier. Il permet de générer automatiquement des devis/factures pour chaque ressource ou services suivant les réservations ou les commandes. Platform-Manager est construit de manière modulaire permettant ainsi l'intégration de fonctionnalités complémentaires aux fonctionnalités de base. Par exemple des modules de gestion de stock, de gestion d'anticorps, de partage de fichiers, de téléchargement de données issues des ressources ont été développés.

Fonctionnement de Platform-Manager

Platform-Manager est construit en 2 niveaux : un niveau d'administration global et un niveaux constitué d’espaces.

  • Administration : le module d'administration permet à l'administrateur système de configurer et de faire les mises à jour de Platform-Manager. Ce module contient également de configurer Platform-Manager : installation et configuration d’espaces, logo, contenu page d'accueil…
  • Espaces : Platform-Manager est constitué "d'espaces" indépendants. Chaque espace est un domaine publique ou privé dans lequel des modules sont activés. Ainsi, un ensemble de plates-formes peuvent partager une seule instance de Platform-Manager avec chacune leur espace dédié. Les outils accessibles dans les "espaces" sont issus des modules dont certains sont interdépendants. Se rapporter à la documentation des modules
